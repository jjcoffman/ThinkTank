Introduction

Scope and Purpose This manual shall be used for grasping an adequate understanding on how to use the Cichlid Simulator software on a computing device.

Simulator Overview

The software shall be executable onto a computing device with Java installed. After the software has been loaded, please refer to the instructions below to learn how to navigate through the main menu, to use objects in the simulation, the camera controls once in the simulator, and how to save scenarios.

First Time Setup

You may choose to run the simulator off a directory of your choosing. You may choose to run it off your hard drive, flash drive, or the school’s network drive . 1.3.1 Compressed (.zip) File If you are presented the Cichlid Simulator in a compressed (.zip) file, extract it to your preferred directory, then run the extracted .jar to begin the simulation. 1.3.2 Creating Necessary File Folders Run the .jar for the first time, and the program will create two folders in the same directory of itself. Scenarios You can drag and drop Scenario files to this folder. Scenarios in this folder will appear in the “Load Scenario” menu. Settings The horizontal resolution determines the width of the simulator (in pixels) the vertical resolution determines the height of the simulator (in pixels) VSync (on or off) These resolution settings should correspond to a preferred aspect ratio(such as 16:9 or 4:3). The default for these values for horizontal and vertical resolution are 640 and 480 respectively, and VSync is on.
